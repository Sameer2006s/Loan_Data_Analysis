📊 Loan Data Analysis Dashboard (Power BI)
🚀 Project Overview

This project presents an interactive Power BI dashboard designed to analyze loan data, uncover financial risk patterns, and provide actionable insights into loan approvals and defaults.

The dashboard focuses on loan distribution, applicant demographics, and risk metrics, helping stakeholders better understand lending behavior and potential default risks.

📌 Key Features
📈 Interactive Power BI dashboard
🔍 Drill-down analysis by demographics, income, and credit score
📊 Loan distribution across multiple categories
⚠️ Risk analysis including default trends and year-over-year changes
📂 Dashboard Tabs (Pages)

This Power BI report is divided into 5 interactive tabs:

🔹 1. Loan Overview & Summary
Loan amount distribution by purpose (Home, Business, Education, Auto, etc.)
Average income by employment type
Average loan amount across age groups

👉 Key Insight:

Loan distribution is fairly balanced (~6500M across categories)
Income remains consistent across employment types (~82K avg)
Unemployed applicants show slightly higher default rates
🔹 2. Default Analysis
Default rate (%) by employment type
Default trends over years (2013–2018)

👉 Key Insight:

Default rates fluctuate around ~11–12% yearly
Employment status impacts default probability
🔹 3. Applicant Demographics
Loan distribution by:
Age group
Marital status
Education level

👉 Key Insight:

Loan distribution is stable across demographic groups
Bachelor's degree holders have the highest number of loans
🔹 4. Financial Profile
Loan amount by credit score category
Loan distribution for adults & middle-aged groups
Mortgage/dependent analysis

👉 Key Insight:

Medium & Low credit score categories contribute the largest loan volumes
Loan patterns are consistent across marital status and age groups
🔹 5. Financial Risk Metrics
Year-over-Year (YoY) Loan Amount Change
YoY Default Loan Change
Loan distribution by income bracket & employment

👉 Key Insight:

Loan growth and defaults show noticeable fluctuations over time
High-income group contributes the largest share (~32.58bn total loan amount)
